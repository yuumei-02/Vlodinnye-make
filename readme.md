# About
Vlodinnye make (or Vmake for short), is a simple build system in C that allows
you to write build recipies in pure C instead of using a whole other language which can be quite annoying.

# Dependencies
As per usual, this project depends on my Mektova-C-Utils library which you can find here: https://github.com/yuumei-02/Mektova-C-Utils
